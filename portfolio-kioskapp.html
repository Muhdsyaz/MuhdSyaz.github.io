KioskApplication

A simple food ordering system.

Kiosk App is a program developed using Eclipse as IDE and all the data are stored on phpMyAdmin. Kiosk App basically is a food-serving application where user can choose and buy their prefered meals. The user interface for this program is inspired by McDonald Website as we are assigned by our lecturer to create a food-serving program with McDonald's menu. This program implemented with TCP/IP client-server to handle data transfer.

  • Menu

    Picture above is the first interface that will be prompted out after the user has login to the system successfully. User can choose the meals that they prefered here. After that, user can click on 'Proceed to Cart' to proceed.

  • Add to cart

    Here, the user can see the list of items that they ordered and the total price of the order. User then click on 'Payment' button continue.

  • Payment

    User then need to insert their valid credit number and click on 'Confirm' button to complete their transaction. The order then send to server.

  • Kitchen

    The server send the order to kitchen client. The kitchen can display the list of ordered item.
