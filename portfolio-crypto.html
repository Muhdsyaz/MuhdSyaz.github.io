CryptocurrencyPrediction

A simple cryptocurrency prediction using machine learning.

ABSTRACT
Cryptocurrency is a decentralized digital exchange system that uses cryptography for secure transactions. Mining, a verification process, ensures transaction authenticity and prevents double spending. Different cryptocurrencies employ various mining technologies to meet their specific needs, such as transaction speed or resource efficiency.

DATA SET
This project focuses on cryptocurrency prediction using one-year datasets for different currencies. The dataset includes six main features: close price, open price, high price, low price, adjusted close price, and volume. Specifically, for Ripple (XRP), the dataset spans from April 15, 2020, to April 15, 2021. Ethereum (ETH) dataset covers May 11, 2020, to May 11, 2021, while Bitcoin (BTC) dataset covers May 18, 2020, to May 18, 2021.
