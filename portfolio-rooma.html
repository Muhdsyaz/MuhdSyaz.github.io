Rooma Application

A platform for rental property advertisement.

Rooma Application is a mobile application which perform as an online marketplace that assists individuals in finding an affordable home to rent. This system allows users to create, manage and update their accounts. This system enables the property owner to post advertisements for their properties and customers can view the details of properties. Customer can directly contact the house owner via WhatsApp and email if they interested in renting the property. Google API will be bought and merged in this system. It will show the exact property location on the maps. The system enables user to be a property owner and a customer at the same time where the user can post an advertisement of property they want to rent out and also, they can find a property to rent.

The expected outcome of the project is the system are able to help property owner to manage their advertisement easily. Besides, the user can use all the functions provided by the system according to their need. The property owner can easily post an advertisement. The customer can use all the details of advertisements provided by the system to find a suitable rental property. Thus, fasten the process of getting a rental property for rent. The system also expected to provide an exact location of the rental property.
